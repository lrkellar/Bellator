# Bellator Properties LLC Website

A professional website for Bellator Properties LLC, a real estate private equity firm.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: Smooth scrolling navigation, animated counters, and hover effects
- **Contact Form**: Functional contact form with validation
- **SEO Optimized**: Proper meta tags and semantic HTML structure

## Sections

- **Hero**: Eye-catching landing section with company mission
- **About**: Company overview with key statistics
- **Services**: Core business offerings and capabilities
- **Portfolio**: Showcase of select property investments
- **Contact**: Contact information and inquiry form

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript
- Google Fonts (Inter)

## Setup

1. Clone or download the repository
2. Open `index.html` in a web browser
3. For GitHub Pages hosting:
   - Push to a GitHub repository
   - Enable GitHub Pages in repository settings
   - Select source branch (usually `main` or `gh-pages`)

## Customization

- Update company information in `index.html`
- Modify colors and styling in `styles.css`
- Replace placeholder content with actual company data
- Add real property images to the portfolio section
- Update contact information and form handling
